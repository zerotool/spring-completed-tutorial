# sprint-websocket-chat-project

Java - 11

Maven - 3.x.x

## Steps to Setup

**1. Clone the application**

**2. Build and run**
```bash
mvn package
java -jar target/...-SNAPSHOT.jar
```
OR
```bash
mvn spring-boot:run
```
